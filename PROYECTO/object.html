Destructuring objects

Ejemplo descriptivo


            
              
//Object Destructuring
              
 let person = {
    name: 'Elon Musk',
    occupation: 'CEO'
  };

  let { name, occupation } = person;

  // name = 'Elon Musk'
  // occupation = 'CEO'


//Object Destructuring en los Parámetros de un Método
  function printOccupation({ occupation }) {
    console.log(occupation);
  }

  let person = {
    name: 'Elon Musk',
    occupation: 'CEO'
  };

  // will print 'CEO' to the console.
  printOccupation(person);

Descripcion

La desestructuración funciona también en objetos. Objeto Destructing se puede utilizar para extraer variables por nombre de clave de objeto.

Características

  • Puede utilizarse en todas partes, desde las declaraciones de importación hasta las declaraciones de parámetros de una función.
  • Se puede desestructurar las claves de objeto en los parámetros de un método, esto permite extraer explícitamente las partes que se necesita para realizar alguna acción, de una manera más obvia y legible.
