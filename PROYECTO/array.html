Destructuring arrays

Ejemplo descriptivo


            
              
//ES5
var arr = [1, 2, 3, 4];
var a = arr[0];
var b = arr[1];
var c = arr[2];
var d = arr[3];
let [a, b, c, d] = [1, 2, 3, 4];

//ES6
console.log(a); // 1
console.log(b); // 2





//Omitir asignaciones con Array Destructuring
let arr = ["Gaius", "Julius", "Caesar", "Octavianus"];

  // pull out only the 3rd element
  let [, , x] = arr;

  // x="Caesar"
  
  
//Clonando Arrays
  let arr = ["Marcus", "Vipsanius", "Agrippa"];

  // clone the array
  let [...copy] = arr;

  // copy = ["Marcus", "Vipsanius", "Agrippa"]
  
  //I
  
    let arr = [5, 6, 7, 8, 9];

  // immutable version of array.shift (we make a copy of the array we remove first element of)
  let [first, ...arrCopy] = arr;

  // first = 5
  // arrCopy = [6, 7, 8, 9]

Descripcion

ALa desestructuración nos permite extraer valores de arrays y objetos (incluso profundamente anidados) y almacenarlos en variables con una sintaxis más conveniente.

Características

  • Se pueden realizar asignaciones de manera muy fácilm solamente poniendo comas donde no se quiere realziar la asignación.
  • La desestructuración también le permite trabajar en conjunto con el operador de propagación. El operador de propagación es una sintaxis de "..." que permite que las expresiones sean expandidas donde se podrían esperar múltiples elementos.
  • Podemos declarar variables con let en lugar de var si no queremos que sean accesibles más allá de un ámbito.
